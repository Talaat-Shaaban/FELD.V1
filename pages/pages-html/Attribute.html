Name Attribute Description
class A space-separated list of the classes of the element.
Classes allow CSS and JavaScript to select and access
specific elements via the class selectors or functions
like the method Document.getElementsByClassName().
id Defines a unique identifier (ID) which must be unique in
the whole document. Its purpose is to identify the element
when linking (using a fragment identifier), scripting, or
styling (with CSS).
style Contains CSS styling declarations to be applied to the
element. Note that it is recommended for styles to be
defined in a separate file or files.
